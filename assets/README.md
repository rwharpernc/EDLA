# Assets Directory

**Last Updated:** 2025-02-04

This directory contains application assets such as icons, images, and other resources.

## Required Files (for executable build)

- `app.ico` - Application icon for Windows executable (256x256 recommended)

## Optional Files

- `splash.png` - Splash screen image (optional)
- Other images/icons as needed

## Notes

- Icons should be in ICO format for Windows
- Use 256x256 or higher resolution for best quality
- ICO files can contain multiple sizes (16x16, 32x32, 48x48, 256x256)
